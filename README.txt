Selamat datang di README Website "Beasiswa HAM x ITTP."

Deskripsi Program:

"Beasiswa HAM x ITTP." merupakan sebuah platform informasi beasiswa yang bertujuan untuk memberikan informasi terkini tentang jenis beasiswa yang tersedia bagi mahasiswa.
Dengan menggunakan website ini, pengguna dapat mengetahui definisi beasiswa secara detail dan rinci.

Fitur Utama:

1. Halaman Utama:
   Halaman ini menampilkan pilihan beasiswa yang tersedia dengan informasi singkat.
   Pengguna dapat mengscroll ke bawah untuk melihat lebih banyak card jenis beasiswa.

2. Halaman Daftar:
   Halaman ini memungkinkan mahasiswa untuk mengajukan pendaftaran beasiswa.
   Pengguna dapat melakukan pendaftaran beasiswa jika memiliki IPK >=3 dan diharuskan melengkapi formulir pendaftaran serta mengunggah berkas persyaratan dalam bentuk pdf, jpeg, jpg, png, zip.
   Namun jika memiliki IPK <3, pengguna tidak bisa memilih jenis beasiswa dan menginputkan berkas syarat serta button daftar akan otomatis terdisable oleh sistem, sehingga pengguna tidak bisa mendaftar program beasiswa.

3. Halaman Hasil:
   Pada halaman ini, pengguna dapat melihat daftar pendaftar beasiswa beserta status verifikasi dalam bentuk tabel.

Cara Menggunakan:

1. Instalasi:
   - Pastikan Anda memiliki server web (misalnya, XAMPP) yang mendukung PHP dan MySQL.
   - Buatlah database baru pada server Anda dan import skema yang disediakan di dalam file SQL.

2. Konfigurasi Koneksi Database:
   - Buka file "koneksi.php".
   - Ubah parameter koneksi database (host, username, password, "nama database") sesuai dengan konfigurasi server Anda.

3. Halaman Utama:
   - Buka file "index.php" pada browser untuk melihat daftar pilihan beasiswa.

4. Halaman Daftar:
   - Klik "Daftar" pada menu navigasi untuk mengakses halaman pendaftaran.
   - Isi formulir pendaftaran dengan informasi yang diminta oleh sistem.
   - Unggah berkas yang diperlukan dan klik "Daftar" untuk mengirimkan formulir.

5. Halaman Hasil:
   - Klik "Hasil" pada menu navigasi untuk melihat daftar pendaftar beasiswa dalam bentuk tabel.
